- CSS Inline
Uma folha de estilo inline, quando as regras CSS são declaradas dentro da tag de abertura do elemento HTML com uso do atributo style. Criar um arquivo "exemplo_01_css_inline.html".

Observem que o CSS e colocado dentro das Tag , essa forma de estilo é interesante, mas não muito prática, pois esses estilos servem somente
para este arquivo html e só podem ser usados naquela linha.
