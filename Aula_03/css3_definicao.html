CSS - Cascade Style Sheet - Folha de Estilo

CSS 3 definição

Até então, trabalhamos apenas com a estrutura dos documentos web, o que influenciava no layout do mesmo, agora iremos nos deter no design, cores, tipografia, alinhamento dos elementos, tamanho das imagens, etc.

Cascade Style Sheet, traduzindo folha de estilos em cascata, comumente conhecido pela sigla CSS, é uma linguagem que aplica formatações de estilo e de layout aos documentos web.

Através do CSS, criou-se o conceito que prioriza o uso dos elementos HTML exclusivamente para marcar e estruturar o conteúdo do documento, não podendo ter nenhum atributo para alteração visual, apenas para parte estrutural da página.

Efeito Cascata do CSS O CSS trabalha sobre um efeito em cascata,que nada mais é, do que regras de prioridades de formatações à um elemento. Quando um estilo é aplicado, e após há alteração do valor do mesmo, o CSS estabelece uma ordem para definição do estilo, priorizando:

1. Declarações do usuário com !important; 2. Estilo inline (dentro de um elemento HTML); 3. Estilo incorporado (definido na seção head do documento); 4. Estilo externo (importado ou linkado)

Tipos de Formatações CSS 26 Existem três formas de formatar os elementos HTML usando o CSS, são elas:

1. Código CSS inline
2. Código CSS interno ou Incorporado
3. Arquivo CSS externo
