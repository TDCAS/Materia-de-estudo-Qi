Links

Links em inglês significa ligação, elo, conexão. Os links estão presentes em todas as páginas Web. Eles permitem a navegaçã de página em página através de um clique. Os mesmos podem ser uma PALAVRA , uma frase ou uma imagem que o usuário clica para ir para outra página, imagem , um e-mail , um enereço de FTP, etc.
Portanto, um link é um endereço, uma referencia de um local para outro.
Para Definir um link , basta utiliza a tag href , no qual define o destino do link.
-Link Externo
Os links externos direcionam o usuário para um documento que não faz parte do site, o mesmo deve ser indicado com protocolo adequado:

Link âncora

Os links âncoras direcionam o usuário para um ponto especifico dentro do documento, normalmente são utilizados fazer links que voltam ao topo da página, ou direciona o usuário à algum subtítulo da página

Nesse caso o valor do href de ser "#"seguido pelo ID do elementos da página qual será direcionada.
